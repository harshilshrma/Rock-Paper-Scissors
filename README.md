# Features

The game has the following features:

<ul>
    <li>
        <strong>Interactive Icons:</strong> Icons for rock, paper, and scissors with dynamic animations on hover, including enlargement, rotation, and boundary addition for a visually stunning effect.
    </li>
    <li>
        <strong>Sound Feedback:</strong> Enjoy a satisfying sound played on an icon-click, enhancing the overall gaming experience.
    </li>
    <li>
        <strong>Color Palette:</strong> An aesthetically pleasing and interactive colour palette that elevates the game's visual appeal.
    </li>
    <li>
        <strong>Real-time Score Updates:</strong> User and computer scores are displayed after every play, keeping players informed about their progress.
    </li>
    <li>
        <strong>Game Outcome Display:</strong> Instantly see whether you won or lost with a text highlight showcasing the game's moves and results.
    </li>
</ul>
